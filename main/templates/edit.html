Custom Lua Functions

Lua 5.4 Reference Manual

FunctionDescription
clear_display()Clear the entire display
set_pixel(x, y, r, g, b)Set a single pixel at (x,y) with RGB color (0-255)
fill_rect(x, y, w, h, r, g, b)Fill a rectangle at (x,y) with width w, height h
draw_hline(x, y, len, r, g, b)Draw a horizontal line starting at (x,y)
draw_vline(x, y, len, r, g, b)Draw a vertical line starting at (x,y)
draw_line(x0, y0, x1, y1, r, g, b)Draw a line from (x0,y0) to (x1,y1)
draw_circle(cx, cy, radius, r, g, b)Draw a circle outline centered at (cx,cy)
draw_filled_circle(cx, cy, radius, r, g, b)Draw a filled circle centered at (cx,cy)
draw_triangle(x0, y0, x1, y1, x2, y2, r, g, b)Draw a triangle outline with 3 vertices
draw_filled_triangle(x0, y0, x1, y1, x2, y2, r, g, b)Draw a filled triangle with 3 vertices
draw_string(str, x, y, r, g, b, [size])Draw text at (x,y). Size: 3, 5, 8 (default), or 16
millis()Returns milliseconds since boot
delay(ms)Pause execution for ms milliseconds

Notes

  • All coordinates are in pixels (x, y)
  • RGB color values range from 0-255
  • Font sizes: 3x3, 5x5, 8x8 (default), 16x16
